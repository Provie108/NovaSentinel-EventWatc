##  **Project Overview**

NovaSentinel EventWatch is a **real-time form submission pipeline** designed for automated alerting, monitoring, and event tracking.

It consists of:

### ‚úî A custom-designed web form  
- Dark mode + neon-purple UI

- Modern UX interactions

### ‚úî A webhook-driven backend

### ‚úî Instant Telegram notifications
